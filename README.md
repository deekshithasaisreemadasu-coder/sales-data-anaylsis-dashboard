## Project Overview

This project analyzes over 50,000 retail sales records to identify revenue trends, regional performance, and profitability using SQL and Excel.

The objective is to transform raw sales data into actionable business insights and an interactive dashboard for decision-making.

---

## Tools Used

* SQL (Data querying & aggregation)
* Microsoft Excel (Pivot Tables & Dashboard)
* Data Cleaning & Analysis

---

## Business Questions Answered

* Which region generates the most revenue?
* Which product categories are most profitable?
* What are the monthly sales trends?
* Are there seasonal patterns in sales?

---

## Key Insights

* The West region generated the highest overall revenue.
* Technology category showed the highest profit margins.
* Furniture category experienced lower profitability.
* Sales dropped approximately 15% in certain regions due to seasonal demand patterns.

---

## SQL Analysis

SQL queries were written to calculate:
